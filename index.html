Welcome to Captain Textbook

The best way to search for free PDFs of expensive university textbooks.


About:

This is a search engine of all the sites with free pdf or epub versions of the textbooks your university wants you to get. Just type the title and author.

If it's online, you'll find it here.

Things to remember:


  1. Do not under any circumstances download any type of .exe file (downloader.exe). It will be a virus.

  2. If you do accidentally begin to download a .exe file, cancel and delete it.

  3. We don't host any textbooks, this engine just searches the web to help you find where files may be.

  4. The file may not be on the first page you try, and it may not be online.

    If that's the case, try some of the sites below to find the cheapest deal.
